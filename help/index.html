Getting Help

Need help with Metron? Try these resources.

Mailing Lists

The best option is to send email to the user list user@metron.incubator.apache.org, which is for user discussions. All of the historic traffic is available in the archive. To subscribe to the user list, please send email to user-subscribe@metron.incubator.apache.org.
